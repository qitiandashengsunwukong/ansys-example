响应谱分析案例：悬臂结构地震响应谱分析
======================================
本例为单点响应谱分析。

集中质量悬臂结构模型，假设其质量集中在三分点处，且 M1=M2=M3=100kg，构件长 3m，截面为 0.15mx0.3m 的矩形面积，材料参数为： 弹性模量 E=3.0E10Pa，泊松比为 0.2。相应的各质量调整为 1000kg。在整个结构上施加如下的加速度响应谱，计算该悬臂结构的 Y 向响应。

| 频率(Hz)    | 响应加速度(m/s^2)  |
|---------- |:-----------------:    |
| 0.3333    |       0.2653          |
| 0.8363    |       0.2653          |
| 1.669     |       0.4941          |
| 2.500     |       0.7108          |
| 3.330     |       0.9210          |
| 4.160     |       1.1260          |
| 5.000     |       1.326           |
| 10.000    |       1.326           |
| 100.000   |       0.6632          |
