独立重力坝的静力分析
====================

![](https://github.com/windstriver/ansys-example/blob/master/STATIC-2D-PLANE182-GravityDam/problem_pic.png)

重力坝坝体很长且两端位移受到约束，而且重力和其它荷载沿长度均匀作用于坝体横截面内，所以坝体的受力状态是典型的平面应变状态。
